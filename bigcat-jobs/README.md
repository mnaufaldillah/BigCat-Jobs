How to Setup:

- Step 1: git clone https://github.com/mnaufaldillah/BigCat-Jobs.git
- Step 2: cd BigCat-Jobs/bigcat-jobs
- Step 3: npm install
- Step 4: npx json-server --watch src/data/db.json --port 8000
- Step 5: open new bash
- Step 6: npm start
